# Exercise 2 - E-Commerce Platform Search Function

## Objective
To implement and compare Linear Search and Binary Search for searching products in an e-commerce platform.

## Product Details
- Product ID
- Product Name
- Category

## Algorithms Used
1. Linear Search
2. Binary Search

## Time Complexity

### Linear Search
- Best Case: O(1)
- Average Case: O(n)
- Worst Case: O(n)

### Binary Search
- Best Case: O(1)
- Average Case: O(log n)
- Worst Case: O(log n)

## Output
Found Product: Watch

## Conclusion
Linear Search is easy to implement and works on unsorted data. Binary Search is faster and more efficient for large datasets when the data is sorted. Therefore, Binary Search is a better choice for e-commerce platforms.
