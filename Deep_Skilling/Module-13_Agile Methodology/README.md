# Module 13 – Agile Methodology

## Overview

This module introduces Agile software development principles and practices. It covers Agile values, Scrum framework, Scrum roles, ceremonies, artifacts, user stories, sprint planning, and Agile project management.

---

## Topics Covered

- Introduction to Agile
- Agile Manifesto
- Agile Principles
- Scrum Framework
- Scrum Roles
- Scrum Ceremonies
- Scrum Artifacts
- User Stories
- Sprint Planning
- Sprint Review
- Sprint Retrospective
- Product Backlog
- Sprint Backlog
- Increment
- Burndown Chart

---

## Learning Outcomes

After completing this module, I was able to:

- Understand Agile software development.
- Explain the Scrum framework.
- Identify Scrum roles and responsibilities.
- Understand Scrum ceremonies.
- Learn Agile artifacts.
- Understand sprint planning and execution.
