# Module 14 – GenAI Fundamentals

## Overview

This module introduces the fundamentals of Generative Artificial Intelligence (GenAI). It covers Large Language Models (LLMs), Prompt Engineering, Retrieval-Augmented Generation (RAG), Embeddings, Vector Databases, AI Agents, and the practical applications of modern AI systems.

---

## Topics Covered

- Introduction to Artificial Intelligence
- What is Generative AI?
- Large Language Models (LLMs)
- Prompt Engineering
- Tokens and Context Window
- Embeddings
- Vector Databases
- Retrieval-Augmented Generation (RAG)
- AI Agents
- Popular Foundation Models
- Applications of GenAI
- Responsible AI

## Learning Outcomes

After completing this module, I was able to:

- Understand the fundamentals of Generative AI.
- Explain how Large Language Models work.
- Learn effective Prompt Engineering techniques.
- Understand Embeddings and Vector Databases.
- Learn the architecture of Retrieval-Augmented Generation (RAG).
- Explore AI Agents and real-world GenAI applications.
- Understand responsible and ethical AI practices.
